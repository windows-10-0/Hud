# Hud
[ESX/QBCore/QBOX] Hud system
# Features
- See Players Health, Hunger, Water, Stamina, Armor level
- Map Type | Circle/Square
- Change map type on /hud menu
- Job hud
- Money hud
- Info hud
# Config
- Disable Info/Money/Job hud
- Server Name
- Speed Type
- Change Refreshing
- Change Getting Fuel function
